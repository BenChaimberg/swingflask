Flask for Swing Paints
======================
Welcome to Ben Chaimberg's Flask attempt. More to come!

####To install (Mac):
First download and unzip file, then:
```
$ cd /path/to/.../flaskexample-master
$ sudo easy_install virtualenv
$ virtualenv venv
$ . venv/bin/activate
$ pip install Flask
$ python flaskexample.py
```
